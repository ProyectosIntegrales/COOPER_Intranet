Guía de utilización de tablas de datos

Una tabla de datos es una manera de representar un conjunto de datos en forma de renglones y columnas las cuales pueden ser solo de consulta o también para administrar los registros mostrados.

Para explicar la utilización de la administración de los datos usaremos como ejemplo la siguiente tabla de empleados la cual contiene columnas de datos de los tipos más comunes que se pueden presentar en nuestro sistema:

En esta tabla se muestran los datos de los empleados de una empresa común donde la primera columna es el Número del empleado, la segunda es el Nombre, la tercera es el Sexo, en la cuarta se indica si el empleado es del tipo administrativo o no.

En la parte superior se muestran los encabezados de las columnas y al extremo derecho tenemos el botón para activar los campos para agregar nuevos registros con el ícono de una crucesita verde .

Como se puede observar, la primera columna sería en este caso un número secuencial generado automáticamente en la entrada de datos.

La segunda columna, el nombre del empleado es del tipo texto y es una entrada manual.

La tercera columna se trata del tipo de selección múltiple el cual en este caso solo contiene dos valores posibles.

La cuatra columna es del tipo booleana (Falso/Verdadero)

Además de las columnas de datos, al lado derecho de la tabla se tienes dos botónes los cuales se utilizan para modificar o borrar los registros.

Al pie de la tabla se muestran los indicadores de página los cuales nos ayudan a cambiar de página mostrando así en cada una de ellas un conjunto distinto de datos.

Agregando registros

Para agregar registros con el cursor del ratón se hace click en la crucecita verde en el encabezado de la tabla para activar los campos en blanco para agregar nuevos registros.

En esta nueva vista, se muestra un renglón adicional con los elementos para crear nuestro nuevo registro. Para la primera columna no se muestra ninguna indicación ya que este dato será agregado automáticamente por el sistema al aceptar crear el nuevo registro.

Para la segunda columna se muestra una caja de texto donde se va a escribir el nombre del empleado. En la tercera columna se muestra una lista de selección en la cual se puede escojere entre las opciones disponibles, en este caso se puede escojer únicamente los datos 'M' o 'F' dependiendo del sexo del empleado. Para la cuarta columna se muestra una caja booleana donde solo si indica si/no según el caso.

Al extremo derecho vemos dos botones representados por los íconos para aceptar o cancelar esta operación. Al dar click sobre el botón para aceptar, se crea el nuevo registro, se oculta este renglón de campos para agregar y el nuevo registro creado será mostrado al final de la lista de datos. En caso de que desee cancelar la operación solo se ocultará el renglón para agregar datos sin crear el nuevo registro.

Modificando registros

Para modificar un registro existente es necesario hacer click con el cursor del ratón sobre el ícono de modificación correspondiente al renglón del registro que se desea modificar.

Al igual que al agregar registros, se habilitan los elementos correspondientes para que cada uno de los diferentes tipos de columnas puedan ser modificados. Para la primera columna no se muestra ninguna indicación y solo se muestra el valor actual ya que este dato no puede ser modificado.

Para la segunda columna se muestra una caja de texto donde aparecel el nombre del empleado actual o dato existente. En la tercera columna se muestra una lista de selección en la cual se puede escojere entre las opciones disponibles apareciendo seleccionada la opción actual del registro. Para la cuarta columna se muestra una caja booleana que puede modificarse su estado existente.

Al extremo derecho del renglón elegido vemos que los dos botones han sido reemplazados por los íconos para aceptar o cancelar esta operación. Al dar click sobre el botón para aceptar, se aplican los cambios al registro, se oculta este renglón de campos para modificar y el registro modificado será mostrado. En caso de que desee cancelar la operación solo se ocultarán del renglón los elementos para modificar datos quedando el registro intacto.

Borrando registros

Para borrar o eliminar un registro existente es necesario hacer click con el cursor del ratón sobre el ícono de la crucesita roja correspondiente al renglón del registro que se desea eliminar.

En este caso solo aparece una caja de diálogo del sistema para confirmar esta operación donde se debe utilizar el botón según se desee.

Al aceptar esta operación es registro se borrará de la tabla y la vista de la tabla se actualizará con el conjunto de datos nuevo. Esta operación no es reversible en ninguno de los casos en cuanto administración de tablas de datos.
